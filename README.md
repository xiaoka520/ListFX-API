## ListFX-API
ListFX的图片API库
## 使用方法
请求`https://api.listfx.top/`即可
## 关于图片
1.图片绝大多数来自[WallHaven](https://wallhaven.cc/)，原图理论上可在该网站内找到<br/>
2.由于图片都是**人类**(不是机器人)挑选出来的，难免会有质量较差、重复的图片，请谅解！如果您认为有此类图片，请您提交issue，附上图片的链接/名称，谢谢🙏<br/>
3.如果我们存储您图片的行为侵犯了您的权益，请提交issue！
